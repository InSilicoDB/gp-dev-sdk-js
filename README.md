# GP Dev SDK for JavaScript

Small JavaScript SDK for the GenePlaza developer API.

## Development

```bash
yarn install
yarn lint
yarn test
```

## Notes

`querySNPGenotypesFromFile()` supports both `.gen.zip` and `.vcf.zip` dataset archives.

## Publish

```bash
git commit -m "<msg>"
npm version <major|minor|patch>
npm publish
```
